This project aims to detect and classify diseases in cotton plants using deep learning techniques. Early detection of diseases is crucial for farmers to take preventive measures and ensure better crop yields. The system uses a Convolutional Neural Network (CNN) to analyze images of cotton leaves and classify them into healthy or diseased categories. 

Dataset and Classes
The model is trained on a dataset comprising 2137+ images of cotton leaves. The system classifies images into the following categories:
